# 주인집 땅따먹기 (turf war)

주사위를 굴려 땅을 최대한 많이 먹는 게임입니다.
(2021.01.08 2:14 개발 시작)

🔗 **[게임 플레이하기](https://goodsoo.github.io/turf-war/)**

![메인 화면](readme/0-main.png)

## 게임 설정

플레이어 수(1명 = 알고리즘과 대결 / 2명), 캐릭터(고양이·강아지), 방 크기(10×10 ~ 20×20)를 선택합니다.
